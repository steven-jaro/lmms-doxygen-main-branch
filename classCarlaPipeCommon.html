◆ msgReceived()

virtual bool CarlaPipeCommon::msgReceived ( const char * msg)
protectedpure virtualnoexcept

A message has been received (in the context of idlePipe()). If extra data is required, use any of the readNextLineAs* functions. Returning true means the message has been handled and should not propagate to subclasses.

Implemented in CarlaBridgeFormat, CarlaEngineNativeUI, CarlaExternalUI, CarlaPipeClient2, CarlaPipeServer2, CarlaPipeServerLV2, CarlaPluginDiscovery, DiscoveryPipe, ExposedCarlaPipeClient, ExternalPluginUI, MidiPatternPlugin, NativePluginAndUiClass, XYControllerPlugin, and ZynPipeClient.
