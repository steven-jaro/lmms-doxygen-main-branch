Detailed Description

Attribute list used in IMessage and IStreamAttributes: Vst::IAttributeList

  • [host imp]
  • [released: 3.0.0]
  • [mandatory]

An attribute list associates values with a key (id: some predefined keys can be found in presetAttributes).
