Detailed Description

This is an asynchronous wait queue that allows a thread to wait for a message, either indefinitely or for a specified time.

This is one example of a way that the JackMidiAsyncQueue class can be extended so that process threads can interact with non-process threads to send MIDI events.

XXX: As of right now, this code hasn't been tested. Also, note the warning in the JackMidiAsyncWaitQueue.cpp about semaphore wait resolution.
