Detailed Description

template<typename T, std::size_t N>
class lmms::ArrayVector< T, N >

A container that stores up to a maximum of N elements of type T directly within itself, rather than separately on the heap.

Useful when a dynamically resizeable container is needed for use in real-time code. Can be thought of as a hybrid between std::array and std::vector. The interface follows that of std::vector - see standard C++ documentation.
