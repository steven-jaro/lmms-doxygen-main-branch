◆ jack_set_thread_creator()

LIB_EXPORT void jack_set_thread_creator ( jack_thread_creator_t creator)

#include <thread.h>

This function can be used in very very specialized cases where it is necessary that client threads created by JACK are created by something other than pthread_create().

After it is used, any threads that JACK needs for the client will will be created by calling the function passed to this function.

No normal application/client should consider calling this. The specific case for which it was created involves running win32/x86 plugins under Wine on Linux, where it is necessary that all threads that might call win32 functions are known to Wine.

Set it to NULL to restore thread creation function.

Parameters
creatora function that creates a new thread
