47/**
48 Common base class for Lv2 plugins
49
50 This class contains a vector of Lv2Proc, usually 1 (for stereo plugins) or
51 2 (for mono plugins). Most of the logic is done there, this class primarily
52 forwards work to the Lv2Proc and collects the results.
53
54 This class provides everything Lv2 plugins have in common. It's not
55 named Lv2Plugin, because
56 * it does not inherit Instrument
57 * the Plugin subclass Effect does not inherit this class
58
59 This class would usually be a Model subclass. However, Qt doesn't allow
60 this:
61 * inheriting only from Model will cause diamond inheritance for QObject,
62 which will cause errors with Q_OBJECT
63 * making this a direct subclass of Instrument resp. EffectControls would
64 require CRTP, which would make this class a template class, which would
65 conflict with Q_OBJECT
66
67 The consequence is that this class can neither inherit QObject or Model, nor
68 Instrument or EffectControls, which means in fact:
69 * this class contains no signals or slots, but it offers stubs for slots
70 that shall be called by child classes
71 * this class can not override virtuals of Instrument or EffectControls, so
72 it will offer functions that must be called by virtuals in its child class
73*/
