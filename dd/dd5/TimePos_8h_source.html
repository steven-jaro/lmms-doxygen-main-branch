45/**
46 Represents a time signature, in which the numerator is the number of beats
47 in a bar, while the denominator is the type of note representing a beat.
48
49 Example: 6/8 means 6 beats in a bar with each beat having a duration of one 8th-note.
50*/

54 TimeSig( int num, int denom );

110 // calculate number of frame that are needed this time

113 // Before, step notes used to have negative length. This
