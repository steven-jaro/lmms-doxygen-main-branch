171 /*!
172 * Affects how the label of the knob is rendered.
173 *
174 * The default mode returns false. The height of the label text is taken into account when a new fixed
175 * size is computed for the Knob. When the label text is painted the descent of the font is used to
176 * compute the base line. The default mode returns false.
177 *
178 * Enabling fixed font size rendering mode leads to the following behavior:
179 * * The height of the label is not taken into account when the new fixed height of the Knob is computed.
180 * Instead a fixed size of 10 is added for the label.
181 * * When the knob is painted the baseline of the font is always set to 2 pixels away from the lower side
182 * of the Knob's rectangle.
183 * * The label is always rendered with a size of SMALL_FONT_SIZE.
184 */
185 bool fixedFontSizeLabelRendering() const { return m_fixedFontSizeLabelRendering; }
