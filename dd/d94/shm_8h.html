◆ jack_shm_server_t

typedef struct _jack_shm_server jack_shm_server_t

A structure holding information about shared memory allocated by JACK.

this persists across invocations of JACK, and can be used by multiple JACK servers. It contains no pointers and is valid across address spaces.

The registry consists of two parts: a header including an array of server names, followed by an array of segment registry entries.
