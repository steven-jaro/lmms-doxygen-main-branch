◆ ApplyRunningStatus() [2/2]

jack_midi_data_t Jack::ApplyRunningStatus ( size_t * size,
jack_midi_data_t ** buffer,
jack_midi_data_t running_status = 0 )

Use this function to optimize MIDI output by omitting unnecessary status bytes. This can't be used with all MIDI APIs, so before using this function, make sure that your MIDI API doesn't require complete MIDI messages to be sent.

To start using this function, call this method with pointers to the size and asio::buffer arguments of the MIDI message you want to send, and set the running_status argument to '0'. For each subsequent MIDI message, call this method with pointers to its size and asio::buffer arguments, and set the running_status argument to the return value of the previous call to this function.

Note: This function will alter the size and asio::buffer of your MIDI message for each message that can be optimized.
