Detailed Description

template<typename ValueType>
class CarlaScopedValueSetter< ValueType >

Helper class providing an RAII-based mechanism for temporarily setting and then re-setting a value.

E.g.

int x = 1;
{
CarlaScopedValueSetter setter (x, 2);
// x is now 2
}
// x is now 1 again
{
CarlaScopedValueSetter setter (x, 3, 4);
// x is now 3
}
// x is now 4
