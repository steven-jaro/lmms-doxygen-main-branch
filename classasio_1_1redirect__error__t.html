Detailed Description

template<typename CompletionToken>
class asio::redirect_error_t< CompletionToken >

Completion token type used to specify that an error produced by an asynchronous operation is captured to an error_code variable. The redirect_error_t class is used to indicate that any error_code produced by an asynchronous operation is captured to a specified variable.
