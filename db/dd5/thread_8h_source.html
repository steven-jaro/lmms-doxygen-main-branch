131/**
132 * This function can be used in very very specialized cases
133 * where it is necessary that client threads created by JACK
134 * are created by something other than pthread_create(). After
135 * it is used, any threads that JACK needs for the client will
136 * will be created by calling the function passed to this
137 * function.
138 *
139 * No normal application/client should consider calling this.
140 * The specific case for which it was created involves running
141 * win32/x86 plugins under Wine on Linux, where it is necessary
142 * that all threads that might call win32 functions are known
143 * to Wine.
144 *
145 * Set it to NULL to restore thread creation function.
146 *
147 * @param creator a function that creates a new thread
148 *
149 */
150void jack_set_thread_creator (jack_thread_creator_t creator) JACK_OPTIONAL_WEAK_EXPORT;
